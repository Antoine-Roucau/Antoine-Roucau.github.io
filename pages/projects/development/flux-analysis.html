À propos du projet

Flux Analysis est un outil complet de surveillance et d'analyse des vulnérabilités de cybersécurité, développé pour suivre en temps réel les alertes et bulletins de sécurité publiés par l'Agence Nationale de la Sécurité des Systèmes d'Information (ANSSI) et d'autres sources fiables.

Dans un contexte où les menaces informatiques évoluent rapidement, il est essentiel pour les professionnels de la sécurité de disposer d'outils permettant de surveiller, centraliser et analyser efficacement les informations sur les vulnérabilités récentes. Flux Analysis répond à ce besoin en automatisant la collecte de données et en fournissant des visualisations analytiques qui facilitent l'identification des tendances et la priorisation des actions de sécurité.

Fonctionnalités principales

  • Extraction automatisée des bulletins de sécurité de l'ANSSI via web scraping
  • Collecte de données CVE (Common Vulnerabilities and Exposures) à partir d'API publiques
  • Analyse des tendances de vulnérabilités par type, sévérité et vecteur d'attaque
  • Génération de rapports et visualisations (graphiques, tableaux de bord)
  • Système d'alerte pour les nouvelles vulnérabilités critiques
  • Base de données locale pour le stockage et l'historisation des informations
  • Interface en ligne de commande pour interroger et filtrer les données
  • Exportation des résultats en différents formats (CSV, JSON, PDF)

Extraction des données de l'ANSSI


import requests
from bs4 import BeautifulSoup
import pandas as pd
from datetime import datetime

class ANSSIScraper:
    """
    Scraper pour les bulletins de sécurité de l'ANSSI
    """
    
    def __init__(self):
        self.base_url = "https://www.cert.ssi.gouv.fr"
        self.alerts_url = f"{self.base_url}/avis/"
        self.headers = {
            "User-Agent": "FluxAnalysis/1.0 Security Research Tool"
        }
    
    def fetch_bulletins(self, limit=None):
        """
        Récupère les bulletins de sécurité de l'ANSSI
        
        Args:
            limit: Nombre maximal de bulletins à récupérer (None = tous)
            
        Returns:
            DataFrame contenant les bulletins de sécurité
        """
        try:
            # Récupérer la page des avis
            response = requests.get(self.alerts_url, headers=self.headers)
            response.raise_for_status()
            
            # Parser le contenu HTML
            soup = BeautifulSoup(response.content, 'html.parser')
            
            # Trouver tous les bulletins
            bulletin_items = soup.select('.cert-alert-item')
            if limit:
                bulletin_items = bulletin_items[:limit]
            
            bulletins = []
            
            # Extraire les informations de chaque bulletin
            for item in bulletin_items:
                # Récupérer les informations de base
                title_elem = item.select_one('.cert-title a')
                date_elem = item.select_one('.cert-date')
                
                if title_elem and date_elem:
                    title = title_elem.text.strip()
                    url = self.base_url + title_elem['href'] if title_elem.has_attr('href') else None
                    date_str = date_elem.text.strip()
                    
                    # Convertir la date au format français en objet datetime
                    date = self._parse_french_date(date_str)
                    
                    # Récupérer les détails du bulletin
                    details = self._fetch_bulletin_details(url) if url else {}
                    
                    # Créer l'entrée pour le bulletin
                    bulletin_entry = {
                        'title': title,
                        'date': date,
                        'url': url,
                        **details
                    }
                    
                    bulletins.append(bulletin_entry)
            
            # Créer un DataFrame
            df = pd.DataFrame(bulletins)
            return df
        
        except Exception as e:
            print(f"Erreur lors de la récupération des bulletins: {e}")
            return pd.DataFrame()
    
    def _fetch_bulletin_details(self, url):
        """
        Récupère les détails d'un bulletin spécifique
        
        Args:
            url: URL du bulletin
            
        Returns:
            Dictionnaire contenant les détails du bulletin
        """
        try:
            response = requests.get(url, headers=self.headers)
            response.raise_for_status()
            
            soup = BeautifulSoup(response.content, 'html.parser')
            
            # Extraire les informations détaillées
            details = {}
            
            # Sévérité
            severity_elem = soup.select_one('.severity-level')
            if severity_elem:
                details['severity'] = severity_elem.text.strip()
            
            # Systèmes affectés
            affected_systems = []
            systems_elem = soup.select('.affected-systems li')
            for system in systems_elem:
                affected_systems.append(system.text.strip())
            
            details['affected_systems'] = affected_systems
            
            # CVE IDs
            cve_ids = []
            cve_elems = soup.select('.cve-references a')
            for cve in cve_elems:
                cve_id = cve.text.strip()
                if cve_id.startswith('CVE-'):
                    cve_ids.append(cve_id)
            
            details['cve_ids'] = cve_ids
            
            # Description
            description_elem = soup.select_one('.bulletin-description')
            if description_elem:
                details['description'] = description_elem.text.strip()
            
            return details
        
        except Exception as e:
            print(f"Erreur lors de la récupération des détails du bulletin: {e}")
            return {}
    
    def _parse_french_date(self, date_str):
        """
        Convertit une date au format français en objet datetime
        
        Args:
            date_str: Chaîne de date au format français (ex: '28 janvier 2023')
            
        Returns:
            Objet datetime
        """
        french_months = {
            'janvier': 'January',
            'février': 'February',
            'mars': 'March',
            'avril': 'April',
            'mai': 'May',
            'juin': 'June',
            'juillet': 'July',
            'août': 'August',
            'septembre': 'September',
            'octobre': 'October',
            'novembre': 'November',
            'décembre': 'December'
        }
        
        for fr_month, en_month in french_months.items():
            if fr_month in date_str.lower():
                date_str = date_str.lower().replace(fr_month, en_month)
                break
        
        try:
            return datetime.strptime(date_str, '%d %B %Y')
        except:
            # En cas d'erreur, on retourne la date actuelle
            return datetime.now()

Analyse et visualisation des tendances


import matplotlib.pyplot as plt
import numpy as np
import pandas as pd
from collections import Counter

class VulnerabilityAnalyzer:
    """
    Classe pour analyser et visualiser les tendances des vulnérabilités
    """
    
    def __init__(self, data):
        """
        Initialise l'analyseur avec les données de vulnérabilités
        
        Args:
            data: DataFrame contenant les données de vulnérabilités
        """
        self.data = data
    
    def analyze_severity_distribution(self):
        """
        Analyse la distribution des vulnérabilités par sévérité
        
        Returns:
            DataFrame avec la distribution des sévérités
        """
        if 'severity' not in self.data.columns:
            return pd.DataFrame()
            
        severity_counts = self.data['severity'].value_counts()
        return severity_counts
    
    def analyze_trends_over_time(self, freq='M'):
        """
        Analyse les tendances des vulnérabilités au fil du temps
        
        Args:
            freq: Fréquence pour le regroupement ('D' pour jour, 'W' pour semaine, 
                  'M' pour mois, 'Q' pour trimestre, 'Y' pour année)
        
        Returns:
            DataFrame avec les tendances temporelles
        """
        if 'date' not in self.data.columns:
            return pd.DataFrame()
            
        # S'assurer que la colonne date est au format datetime
        self.data['date'] = pd.to_datetime(self.data['date'])
        
        # Regrouper par période et compter les vulnérabilités
        time_series = self.data.groupby(pd.Grouper(key='date', freq=freq)).size()
        
        return time_series
    
    def analyze_affected_systems(self, top_n=10):
        """
        Analyse les systèmes affectés par les vulnérabilités
        
        Args:
            top_n: Nombre de systèmes les plus affectés à retourner
        
        Returns:
            DataFrame avec les systèmes les plus affectés
        """
        if 'affected_systems' not in self.data.columns:
            return pd.DataFrame()
            
        # Aplatir la liste des systèmes affectés
        all_systems = []
        for systems_list in self.data['affected_systems']:
            if isinstance(systems_list, list):
                all_systems.extend(systems_list)
        
        # Compter les occurrences
        system_counts = Counter(all_systems)
        
        # Convertir en DataFrame et trier
        df_systems = pd.DataFrame({
            'system': list(system_counts.keys()),
            'count': list(system_counts.values())
        })
        
        df_systems = df_systems.sort_values('count', ascending=False).head(top_n)
        
        return df_systems
    
    def visualize_severity_distribution(self, save_path=None):
        """
        Visualise la distribution des vulnérabilités par sévérité
        
        Args:
            save_path: Chemin pour sauvegarder l'image (None = afficher uniquement)
        """
        severity_data = self.analyze_severity_distribution()
        
        if severity_data.empty:
            print("Pas de données de sévérité disponibles")
            return
        
        plt.figure(figsize=(10, 6))
        
        # Palette de couleurs selon la sévérité
        colors = {
            'Critique': 'darkred',
            'Haute': 'red',
            'Élevée': 'red',
            'Moyenne': 'orange',
            'Modérée': 'orange',
            'Basse': 'green',
            'Faible': 'green',
            'Indéterminée': 'gray'
        }
        
        # Assigner des couleurs par défaut pour les sévérités non spécifiées
        bar_colors = [colors.get(sev, 'blue') for sev in severity_data.index]
        
        severity_data.plot(kind='bar', color=bar_colors)
        plt.title('Distribution des vulnérabilités par sévérité')
        plt.xlabel('Sévérité')
        plt.ylabel('Nombre de vulnérabilités')
        plt.tight_layout()
        
        if save_path:
            plt.savefig(save_path)
        else:
            plt.show()
    
    def visualize_trends_over_time(self, freq='M', save_path=None):
        """
        Visualise les tendances des vulnérabilités au fil du temps
        
        Args:
            freq: Fréquence pour le regroupement
            save_path: Chemin pour sauvegarder l'image
        """
        time_data = self.analyze_trends_over_time(freq=freq)
        
        if time_data.empty:
            print("Pas de données temporelles disponibles")
            return
        
        plt.figure(figsize=(12, 6))
        
        time_data.plot(kind='line', marker='o')
        plt.title('Tendances des vulnérabilités au fil du temps')
        plt.xlabel('Date')
        plt.ylabel('Nombre de vulnérabilités')
        plt.grid(True, linestyle='--', alpha=0.7)
        plt.tight_layout()
        
        if save_path:
            plt.savefig(save_path)
        else:
            plt.show()

Défis techniques

  1. Extraction robuste des données

    Un des défis principaux était de développer un système d'extraction de données robuste capable de s'adapter aux changements dans la structure des sites web sources et de gérer les erreurs potentielles lors de la récupération des informations.

  2. Normalisation des données hétérogènes

    Les informations sur les vulnérabilités proviennent de sources diverses avec des formats variés. La normalisation de ces données pour permettre une analyse cohérente a nécessité un traitement sophistiqué et des transformations adaptées.

  3. Analyse contextuelle des vulnérabilités

    Au-delà de la simple collecte des informations, l'outil devait analyser le contexte des vulnérabilités pour déterminer leur impact potentiel et leur criticité, en prenant en compte divers facteurs comme le vecteur d'attaque, les conditions d'exploitation, etc.

  4. Visualisations pertinentes

    Créer des visualisations qui transforment les données brutes en informations actionnables pour les utilisateurs, en mettant en évidence les tendances significatives et les points d'attention prioritaires.

Résultats et apprentissages

Ce projet m'a permis d'approfondir mes connaissances en cybersécurité et en analyse de données :

  • Maîtrise des techniques d'extraction de données web (web scraping) avec Beautiful Soup et Requests
  • Compréhension approfondie des standards de vulnérabilités (CVE, CVSS)
  • Analyse et visualisation de données avec Pandas et Matplotlib
  • Développement d'un système d'alerte basé sur des critères de sévérité

  • Gestion de données temporelles et analyses de tendances

Flux Analysis est maintenant un outil fonctionnel que j'utilise régulièrement pour suivre les nouvelles vulnérabilités et menaces en cybersécurité. Il permet de gagner un temps considérable dans la veille de sécurité et fournit des analyses pertinentes pour prioriser les actions de remédiation.

Utilisation de l'outil

  1. Installation

    
# Cloner le dépôt
git clone https://github.com/Antoine-Roucau/Flux_Analysis.git
cd Flux_Analysis

# Installer les dépendances
pip install -r requirements.txt

  2. Collecte des données

    
# Lancer la collecte des données
python flux_analysis.py collect --source anssi

  3. Analyse des tendances

    
# Analyser les tendances sur les 3 derniers mois
python flux_analysis.py analyze --period 3M

  4. Génération de rapports

    
# Générer un rapport complet
python flux_analysis.py report --format pdf --output rapport_securite.pdf
