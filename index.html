Politique de Confidentialité de l'Extension Oups Mail

L'extension "Oups Mail" ne collecte aucune donnée personnelle des utilisateurs.

Utilisation des Autorisations

Cette extension utilise les autorisations suivantes pour fonctionner correctement :

  • activeTab : Utilisé pour accéder à l'onglet actif et interagir avec le contenu de manière contextuelle.
  • storage : Utilisé pour stocker des données localement sur l'appareil de l'utilisateur.
  • scripting : Utilisé pour exécuter des scripts sur les pages web, nécessaire pour interagir avec les fonctionnalités spécifiques de Gmail.
  • https://mail.google.com/ : Utilisé pour accéder au domaine Gmail afin d'intercepter l'envoi d'e-mails.
